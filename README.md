# Jogo da Velha 🕹️

Projeto desenvolvido como parte da avaliação do Curso de Git, fornecido como curso de extensão pela Universidade Estadual do Piauí(UESPI). O objetivo é criar um jogo clássico e funcional utilizando tecnologias web básicas.

## 🚀 Tecnologias
Este projeto utiliza:
* HTML5 para a estrutura
* CSS3 para o design e layout
* JavaScript para a lógica
* Git & GitHub para o controle de versão

## 🎮 Como Jogar
1. Clone o repositório:
   git clone (https://github.com/loizecoutinho/Jogo_da_Velha.git).
2. Abra o arquivo index.html em qualquer navegador (Chrome, Edge, Firefox).
3. Clique nas casas do tabuleiro para jogar contra seu colega localmente.

## 🛠️ Organização do Git
O grupo utilizou a seguinte estrutura de pastas:
/css: Arquivos de estilização.
/js: Lógica do jogo.
index.html: Página principal.
